The one insight

A parimutuel pool is a CRDT.

Everyone “knows” a prediction market needs three centralized things: a matching engine, an oracle, and a custodian. Curva removes all three — because pool totals are sums, and sums don’t care about order.

No server

The swarm matches

Payout is a pure function of (final bet-set, outcome). Merge order doesn’t matter, so partitions heal for free — no consensus, no matching engine. The only thing needing order is the betting cutoff, and Autobase gives exactly that one fence.

No oracle company

The crowd watches

At watch-party scale the oracle problem dissolves: everyone is literally watching the answer. Each device runs on-device ASR over the commentary and the crowd signs a stake-and-writer quorum.

No custodian

Everyone self-settles

Self-custody + parimutuel ⇒ settlement is just netting: compute the minimal transfer set and each loser signs their own USDt transfer from their own wallet. No company ever holds the money.

Polymarket needs AWS, an oracle company, and a custodian.
Curva needs two phones.

What’s proven, not asserted

A CS thesis with tests.

237 tests — property, fuzz, quorum-safety, and a full headless end-to-end — all green in CI.

  • Conservation — Σ payouts + Σ fees ≡ Σ stakes, exactly, for arbitrary bet sets.
  • Commutativity — any permutation, partition, or merge yields identical pools and payouts.
  • Convergence under fuzz — 100 adversarial swarm runs heal to one byte-identical view.
  • Quorum safety — the dual ⅔ threshold makes two outcomes reaching quorum impossible; no whale or sock-puppet can steer.
  • Netting soundness — the transfer set settles every party exactly in ≤ n−1 edges.
  • Serverless by construction — the demo/offline path uses no network at all.

Run it

One command. No network.

The whole thesis, headless — derive vaults → open a terrace → three bets → kill the host → cutoff lock → on-device ASR resolves → net the payout → each debtor settles their own USDt → everyone’s square.

# the whole pipeline, headless, no external services
npm install
npm run demo

# typecheck + lint + 237 tests
npm run check

# the live P2P app (needs the Pear runtime)
npm run build && pear run apps/terrace
